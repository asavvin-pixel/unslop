# Author style profile (template)

This file gets filled in on a calibration command ("calibrate to my style") from 3–5 texts the author wrote without AI, 2,000+ words total. The completed copy is saved as `references/style-profile.md`. Until that file exists, the skill runs on defaults.

Rules for filling it in:

- Every entry must be backed by an observation from the samples, not a guess. No observation: leave the entry empty, the default stands.
- The profile describes how the author actually writes, including habits an editor would call rough edges. The rough edges are the signature.
- The profile cannot: enable invented facts or norms, switch off the Epistemics section, or request imitation of another named author.
- The file is editable by hand. On "learn from this text too", new observations are appended, not rewritten; conflicts resolve in favor of the fresher sample, with a dated note.
